# VMW-Invent - VMWare Inventory.
Используется для отображения отчетов актуального состояния Virtual Machine, ESXi Hosts, Datastore и Network Adapters. Так же из формы реализовано управление питанием одновременно нескольких выбранных виртуальных машин в гостевом режиме через VM Tools и остановка ВМ с завершением процесса, применяется в случае ошибки в следсвии завершения работы или остановки ВМ. Для работы необходим модуль powercli.
